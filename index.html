OPERACIONES BÁSICAS

SUMA

La suma es la operación que combina dos o más números para obtener su total.

Ejemplo: 5 + 3 = 8

Se representa con el signo "+"

RESTA

La resta es la operación que determina la diferencia entre dos números.

Ejemplo: 7 - 2 = 5

Se representa con el signo "-"

MULTIPLICACIÓN

La multiplicación es la operación que combina dos o más números para obtener su producto.

Ejemplo: 4 * 6 = 24

Se representa con el signo "*"

DIVISIÓN

La división es la operación que determina cuántas veces un número (el divisor) cabe dentro de otro número (el dividendo).

Ejemplo: 12 / 3 = 4

Se representa con el signo "/"
